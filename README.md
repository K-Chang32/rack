Convox command line interface.

## Installation

**OS X**

    $ curl -Ls https://convox.s3.amazonaws.com/install/osx.sh | sudo bash

**Golang**

    $ go get -u github.com/convox/cli/convox

## Usage

    $ convox help

## Updates

    $ convox update
